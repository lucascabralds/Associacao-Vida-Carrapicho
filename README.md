# Associação Vida Carrapicho 🌻

Este repositório contém o código-fonte do sistema web desenvolvido para a **Associação Vida Carrapicho**, uma ONG dedicada a transformar a vida de crianças e famílias em situação de vulnerabilidade.

O projeto está dividido em duas partes principais: o site institucional (Landing Page) e o sistema de gestão (Painel Administrativo).

## 📁 Estrutura do Repositório

### 1. Frontend Institucional (Pronto)
A raiz do projeto (arquivos `index.html`, pasta `src/` e pasta `img/`) contém a Landing Page finalizada da ONG. Esta é a vitrine principal do site, focada em apresentar a associação para o público.

* **`index.html`**: Estrutura principal da página inicial.
* **`src/`**: Contém todos os estilos CSS e a lógica JavaScript (incluindo o modal de doações PIX dinâmico e o gráfico de transparência).
* **`img/`**: Todas as imagens, ícones e logotipos utilizados no site.
* **Status**: ✅ Finalizado (Pronto para apresentação).

### 2. Protótipo do Painel Administrativo (Em Desenvolvimento)
A pasta **`Projeto-carrapicho/`** contém o protótipo do sistema interno da ONG. Esta área é restrita e focada na gestão administrativa.

* **O que inclui**: Desenvolvimento dos sistemas de **Login**, autenticação e os **Painéis de Gestão** (Eventos, Transparência, Usuários).
* **Tecnologias**: Envolve a separação entre backend e frontend voltados para as rotas da aplicação administrativa.
* **Status**: 🚧 Em desenvolvimento.
